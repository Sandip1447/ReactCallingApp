# ReactCallingApp
Make a phone call with location base. Get current location 

## Details

Created with **npx create-expo-app**.

Packages used:

- **Expo**

### with Expo

- Clone this repo `git clone https://github.com/Sandip1447/ReactCallingApp.git`
- `cd ReactCallingApp`
- run `npm install`
- run `npx expo start`
